Atari Centipede Remake

This is a remake of Atari Centipede using C++ and a basic 2D video game engine.

Gameplay

This video showcases gameplay for the re-creation. I grabbed sprites for the the game online and played the original Centipede game to understand the gameplay mechanics. For general implementation details, I used design patterns for many different systems. The factory pattern was used to spawn different enemies and an object pool to store different destroyed entities. A singleton pattern was used for different types of managers.
