Donut Boy 3

Donut Boy 3, created by igamebank, was originally released in 1998 on the web using the Shockwave plug-in (Flash precursor). This game no
longer runs on modern-day web browsers, so I was tasked with re-creating the game to run on modern-day web browsers.

Gameplay Trailer

This video showcases gameplay for the re-creation. I was given no source code to reference, so I had to recreate the game mechanics from sight. The biggest implementation challenge was getting the physics to feel right. I developed this using Unity/C#.
