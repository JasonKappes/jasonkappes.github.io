Additional Technical Projects

• Graphics Package • This was a 3 month class project using a high-level graphics API (DirectX 10) to create a graphics package. I gained an understanding of the rendering pipeline and implementing basic shaders, texture mapping, and environment mapping.

• Multithreaded Audio Application • This was a multithreaded application that replayed the Jetson's (Hit 60's TV show) theme song using 24+ threads. There were multiple class-specific rules that made the assignment more difficult. I learned more about basic multithreading implementation using mutexes and futures.

• Custom C++ Memory System • This was an application that created a custom memory de/allocation system. A set amount of space was given where memory blocks were free/used based on how much memory was allocated. This memory system proved to be faster than the standard memory allocation system.
